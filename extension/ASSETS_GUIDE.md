### Screenshot 4: Full AI Analysis
**What to capture**: Complete analysis with all features
- Show lead score (0-100)
- Show priority level (High/Medium/Low)
- Show insights list
- Show personalized recommendations
- Make it look impressive

### Screenshot 5: Limit Modal / Pricing
**What to capture**: Subscription tiers or usage limit modal
- Either capture the limit reached modal
- Or the pricing page (pricing.html)
- Show clear pricing tiers
- Highlight value proposition

### How to Take Screenshots
1. Set browser window to 1280x800 resolution
2. Use browser screenshot tool or:
   - Windows: Windows + Shift + S
   - Mac: Cmd + Shift + 4
3. Save as PNG format
4. No text overlays needed (Chrome Store adds those)
